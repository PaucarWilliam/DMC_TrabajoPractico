# Proyecto Módulo 1 – Fundamentos de Programación

## Descripción

Aplicación interactiva desarrollada en Streamlit que integra los conceptos fundamentales de programación aprendidos en el Módulo 1 de la Especialización en Python for Analytics.

El proyecto demuestra el uso de:

- Variables  
- Condicionales  
- Listas y diccionarios  
- Funciones  
- Programación funcional (map y lambda)  
- Programación Orientada a Objetos (POO)  
- Visualización de datos con Pandas  

La aplicación permite registrar actividades financieras, evaluar presupuestos, calcular retornos esperados y modelar actividades mediante clases.

## Instrucciones de Ejecución

1. Clonar el repositorio  
2. Instalar dependencias:

```
pip install -r requirements.txt
```

3. Ejecutar la aplicación:

```
streamlit run app.py
```
